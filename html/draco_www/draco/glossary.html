Glossary

This page contains definitions of some of the terms in use in Draco. Alternative definitions of some of these terms are possible, and in fact are in use elsewhere. Consequently, we find it helpful to clarify our usage with this glossary. This prevents confusion and facilitates discussions within the Draco development and client community.
Component
Usually synonymous with product, but can be context dependent.

Package
Synonymous with product.

Product
Anything that can be produced from a source code tree.

Program
An umbrella for one or more systems.

Project
An undertaking that has a definite beginning and ending date and produces a product.

System
A product or group of products that persist over time.
