Procedures Archive

This page provides a listing of documents which describe the development, operation and maintenance of the Draco CPS. These documents are available to Draco developers and users through CVS checkout of the "draco_doc" module. Documents may be identified via the Draco Document Number (DDN) for convenient reference.

Document Title & Links DDN Author(s) Description
"The GNATS Bug Tracking System" 99-?? McGhee Describes how to setup and use the GNATS bug tracking system.
"Draco CVS Repository" 99-?? Roberts Describes the operation of the Draco CVS code repository. (under development)
"XEmacs Development Environment for Draco" 99-?? Evans Describes the Draco Elisp code development environment.
"Draco Release Policy and Procedures" 99-?? Evans Standards for the tagging and release of Draco and Draco components.
"The Draco Build System"
99-?? Evans, Roberts Describes the design and operation of the Draco build system.
"Generic Programming in the SOLON Interface" 99-?? Roberts Describes the use of generic programming in the SOLON diffusion package interface. This is an example of the recommended interface approach for Draco packages.
"Multi-Material-Cell Material Properties Using a Temperature-Equilibration Treatment" 99-?? McGhee Describes the principles used in the construction of the Multi-Material cell material properties class.
"Specification of the MT Concept" 99-?? Pautz Describes the Draco XYZ Mesh Type (MT) as a "Concept" in the STL sense.
"Using Purify in Draco with MPI" 99-?? Evans How to use the Purify error detection utility for Draco packages.
"The Draco System for Transport Code Development" 99-?? Evans Describes the principles behind the organization of the Draco code repository, build system, and world-view.
"The Draco 1999 Project Charter" 99-?? McGhee The stated goals of the Draco project for the year 1999.
"Expression Engine Traits" 99-?? Roberts Describes a technique for automatically deciding whether or not to glom expression engine traits onto a container class.
Doxygen C++ Automatic Documentation System 99-?? Evans and McGhee Describes standards and procedures for the use of the Doxygen automatic code documentation system within Draco.
