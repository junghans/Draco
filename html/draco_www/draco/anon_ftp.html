This page describes the procedures for using the Draco anonymous FTP site.

Copying Files from the FTP Site

To get a copy of file "filename" from our site use the following procedure:
  • ftp portal.lanl.gov
  • user name: anonymous@laws.lanl.gov
  • password: your e-mail address
  • cd pub/radtran
  • bin (this step should be skipped if copying an ASCII file)
  • get filename
  • quit

Putting Files on the FTP Site

To put a copy of file "filename" on our FTP site use the following procedure:
  • ftp portal.lanl.gov
  • user name: anonymous@laws.lanl.gov
  • password: your e-mail address
  • cd pub/incoming
  • bin (this step should be skipped if sending an ASCII file)
  • send filename
  • quit

Copying Files to the FTP Site
(Draco Team Members)

If you a Draco team member working from the LANL LAN and wish to place a file on our FTP site for an external user to pick up, use the following procedure to put a copy of file "filename" onto our FTP site:
  • cp filename /n/laws/ftp/ftpuser/pub/radtran/filename
  • chmod o+r /n/laws/ftp/ftpuser/pub/radtran/filename
