Ahtu Problem Specification System

Ahtu is a software system that provides a means to completely specify a radiation transport problem. Components of the specification include:
  • Client specific control parameters
  • Mesh description including vertex coordinates and connectivity
  • Material and radiation volume and boundary sources
  • Material properties
  • State information such as temperatures, velocities, initial conditions and so forth
  • Boundary conditions
  • Edits
This system is currently in the design phase. It will rely heavily on the Python interpreted language for it's implementation. Ahtu is a client of the Draco computational physics system.
